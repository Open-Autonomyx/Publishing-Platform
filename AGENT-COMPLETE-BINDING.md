## 🎯 USING THE SYSTEM

### Question: What do I do right now?
```
1. Check AGENT-MASTER-INDEX.md → "Quick Navigation"
2. Find your situation
3. It points you to specific document section
4. Read that section
5. Execute the procedure
```

### Question: Is it safe to do X?
```
1. Check AGENT-CONSTITUTION.md → Article VI (Safety Guarantees)
2. Check AGENT-HANDBOOK.md → Part 7 (Authorization Table)
3. Verify risk level
4. Decide execute or escalate
```

### Question: How do I run this command?
```
1. Check AGENT-HANDBOOK.md → Part 4 (Command Reference)
2. Find your command
3. See syntax, risk level, when to use
4. Execute with proper authentication
```

### Question: What happened and why did Agent do X?
```
1. Check AGENT-LANGUAGE.md → Operational logs
2. Review log file in ~/.creative-platform/operator-reports/
3. Check decision reasoning in log
4. If unclear, read related section in Constitution/Instructions
```

### Question: Agent exceeded its authority!
```
1. Consult AGENT-CONSTITUTION.md → Article II (Authority)
2. Determine authority level of action
3. Check if prerequisites met
4. If violated: HALT, escalate, restore state
5. Investigate in AGENT-CONSTITUTION.md → Section 2.3
```
